PASDA Harvester

Purpose

This script parses metadata from the PASDA search results page and produces two CSV files:

  • A Primary CSV with all main metadata fields
  • A Distribution CSV with download, landing page, and metadata links

Step 1: Download the PASDA Search Results

Go to the following page and use Save As → Web Page, Complete to save it as pasda-search.html in this project's /data folder.

https://www.pasda.psu.edu/uci/SearchResults.aspx?Keyword=+

Step 2: Run the PASDA Harvester
