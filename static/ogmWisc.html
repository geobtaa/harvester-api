Transform UW-Madison GBL 1.0 Metadata

Purpose: This script reads a batch of GBL 1.0 JSON files and produces two CSV files:

  • A Primary CSV containing all metadata except the link fields
  • A Distribution CSV with only the link fields (e.g., landing page, download, REST services)

This version is tailored specifically for UW-Madison records. The input should be a folder of metadata JSONs located at data/ogmWisc.

GeoBlacklight and OpenGeoMetadata records are often shared as JSON files in bulk. To process these for the BTAA Geoportal, we convert them into a normalized tabular format.
